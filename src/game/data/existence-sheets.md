# 存在シート（Existence Sheets）

修飾語11 × 主語17 = 187存在

- **interpretation**: デフォルトの絵。テンプレートが最初に参照する解釈
- **alt**: 文脈次第で切り替わる別の読み。括弧内は使い分けのヒント
- altは無理に2つ揃えなくてよい。良い解釈が1つなら1つでよい
