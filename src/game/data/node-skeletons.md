## ノード A: 忘れられた橋

- **文構成**: [mod][subj]が 橋を 塞いでいる
- **可変スロット**: mod + subj
- **固定**: 目的語「橋」(を格) + 述語「塞いでいる」
- **パターン数**: (11+1) × (17+1) = 216
  - +1 = 元の修飾語/主語（忘れられた + 何か）
- **元テキスト**: 「忘れられた柱が橋を塞いでいる」
- **元カード**:
  - s1(mod): `orig_a_mod` "忘れられた"
  - s2(subj): `orig_a_subj` "柱"
  - s3(obj): 固定「橋」
  - s4(pred): 固定「塞いでいる」
- **ノードの文脈**: 古い石橋。何かが通行を妨げている。橋自体は頑丈
- **AP**: 3

### 格の注意
- 述語「塞いでいる」は固定なので、格の歪みは発生しない
- 全ての修飾語×主語の組み合わせが有効

## ノード C: 霧の街道

- **文構成**: 呪われた[subj]が 街道を [pred]
- **可変スロット**: subj + pred
- **固定**: 修飾語「呪われた」+ 目的語「街道」(を格)
- **パターン数**: (17+1) × (17+1) = 324
- **元テキスト**: 「呪われた荊が街道を覆っている」
- **元カード**:
  - s1(mod): 固定「呪われた」
  - s2(subj): `orig_c_subj` "荊"
  - s3(obj): 固定「街道」
  - s4(pred): `orig_c_pred` "覆っている"
- **ノードの文脈**: 霧に包まれた街道。何かが通行を阻む。呪いの気配
- **AP**: 3

### 格の注意（重要）
- 目的語「街道」はを格
- を格 + 自動詞/に格動詞 = **歪み**
  - 自動詞/なし: 輝く(pred_shine)、歌っている(pred_sing)、踊っている(pred_dance)、笑っている(pred_laugh)
  - に格動詞: 眠っている(pred_sleep)、絡みついている(pred_entangle)、降り注いでいる(pred_rain)、囁いている(pred_whisper)、待っている(pred_wait)、溶けている(pred_melt)
  - 「逃げていく」は経路のを格として許容
- 歪みになる述語: 10種 × 18主語 = **180パターンが歪み**
- 有効パターン: 324 - 180 = **144** + 元テキスト1

## ノード D: 泉の声

- **文構成**: [mod]歌が 泉から [pred]
- **可変スロット**: mod + pred
- **固定**: 主語「歌」+ 目的語「泉」(から格)
- **パターン数**: (11+1) × (17+1) = 216
- **元テキスト**: 「澄んだ歌が泉から響いている」
- **元カード**:
  - s1(mod): `orig_d_mod` "澄んだ"
  - s2(subj): 固定「歌」
  - s3(obj_kara): 固定「泉」
  - s4(pred): `orig_d_pred` "響いている"
- **ノードの文脈**: 美しい泉のほとり。歌声が聞こえる。修飾語で歌の性質が変わり、述語で何が起きるかが変わる
- **AP**: 3

### 格の注意（重要）
- 目的語「泉」はから格
- から格 + 他動詞 = **歪み**: 塞いでいる(pred_block)、守っている(pred_guard)、追いかけてくる(pred_chase)、隠している(pred_hide)、砕く(pred_crush)
  - 見つめている: 「泉から見つめている」→ OK（発信源）
  - 囁いている: 「泉から囁いている」→ OK（発信源）
- 歪みになる述語: pred_block, pred_guard, pred_chase, pred_hide, pred_crush → 5種 × 12修飾語 = 60パターンが歪み
- 有効パターン: 216 - 60 = **156** + 60歪み

## ノード E: 王の間

- **文構成**: [mod][subj]が [adv] 玉座を 守っている
- **可変スロット**: mod + subj + adv（副詞は後付け方式）
- **固定**: 目的語「玉座」(を格) + 述語「守っている」
- **パターン数**: テキスト (11+1) × (17+1) = 216 + 副詞は後付け
- **元テキスト**: 「忠実な衛兵が玉座を守っている」
- **元カード**:
  - s1(mod): `orig_e_mod` "忠実な"
  - s2(subj): `orig_e_subj` "衛兵"
  - s3(obj): 固定「玉座」
  - s4(pred): 固定「守っている」
  - s5(adv): 副詞スロット（後付け）
- **ノードの文脈**: 王の間。玉座を何かが守っている。王はいない
- **AP**: 4（副詞スロットがある分、1多い）

### 格の注意
- 述語「守っている」は固定なので、格の歪みは発生しない
- 全ての修飾語×主語の組み合わせが有効

### 副詞の後付け方式
- テキスト本体は副詞なしで書く（216パターン）
- 副詞が使われた場合、本文の後に定型句を追加:
  - 静かに: 「——静かに、音もなく。」
  - 激しく: 「——激しく、狂ったように。」
  - ゆっくり: 「——ゆっくりと、急ぐ素振りもなく。」
  - 力なく: 「——力なく、今にも崩れそうに。」
  - 突然: 「——それは突然のことだった。」
