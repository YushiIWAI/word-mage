# ノードP13「裂け目」Outcome一覧

- **文構成**: [mod][subj]が 裂け目から 囁いている
- **可変スロット**: mod + subj
- **固定**: 目的語「裂け目」(から格) + 述語「囁いている」
- **ノードの文脈**: 大地の裂け目。底から何かが囁いている。耳を傾けるか、無視するか
- **格歪みなし**（「囁いている」はに/から格自動詞、歪み対象外）
