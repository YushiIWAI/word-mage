# Outcome生成指示書

## この指示書の使い方
ノードごとのoutcome一覧を生成する際、このファイルを丸ごとプロンプトに含める。
ノード固有の情報（文構成、可変スロット、文脈等）は別途与える。

---

## 1. 生成対象

- **ロスターカードの組み合わせのみ**。ロスター外の語（orig語）は存在しない
- 元テキスト概念は廃止済み。初期値=ロスターカード。「書き換えなかった場合」=初期値のまま確定した場合のOutcome
- 可変スロットのデフォルト値は必ずロスター50枚から選出済み

## 2. 格歪みルール

| 目的語の格 | 歪みとなる述語 | 歪み数 |
|-----------|--------------|-------|
| を格 | 眠っている, 絡みついている, 降り注いでいる, 囁いている, 溶けている, 歌っている, 踊っている, 笑っている, 輝く | 9/17 |
| に格 | 塞いでいる, 守っている, 追いかけてくる, 隠している, 砕く, 見つめている | 6/17 |
| から格 | 塞いでいる, 守っている, 追いかけてくる, 隠している, 砕く, 眠っている, 待っている, 絡みついている | 8/17 |
| で格 | 塞いでいる, 守っている, 追いかけてくる, 隠している, 砕く, 絡みついている, 降り注いでいる | 7/17 |

- 「逃げていく」は経路のを格として常に許容

## 3. 出力フォーマット

```
### [修飾語] × [主語]
- **ID**: [ノードID]_[mod短縮]_[subj短縮]
- **conditions**: { s1: '[mod_id]', s2: '[subj_id]' }
- **text**: [outcomeテキスト]
- **damage**: [数値]
- **quill**: [数値]
- **reward**: [なし / rewardItem名(sellPrice)]
```

### IDの短縮ルール
mod: small/gentle/fire/hard/sharp/broken/mighty/frozen/clear/cursed/sleepy
subj: kitten/snake/dragon/butterfly/wind/flame/ice/shadow/song/silence/knight/witch/king/ghost/mirror/doll/rain
pred: block/guard/chase/gaze/hide/crush/flee/wait/sleep/entangle/whisper/melt/shine/rain_pred/sing/dance/laugh

例: `p06_gentle_butterfly` / `p07_ghost_flee` / `p10_wind_guard`

subj+pred可変の場合: `{ノードID}_{subj}_{pred}`
mod+pred可変の場合: `{ノードID}_{mod}_{pred}`

## 4. 文体ルール

### 必須
- **三人称現在形**の寓話体。「〜した」ではなく「〜している」「〜だ」
- 1 outcome = **1〜3文**。最大60文字程度。短い方がよい
- 乾いたトーン。感情の押し売りをしない
- 状況で語る。感情を説明しない（「恐ろしいと感じた」→NG。「背筋が凍る」→OK）

### 禁止
- 「あなた」「プレイヤー」等のメタ呼称
- 「文が歪んだ」等のメタ表現（歪みはdefaultOutcomeの仕事）
- 説明的すぎる描写
- 同じ文型パターンの3回以上連続使用
- 「気がした」「感じた」「思った」等の内面描写

### トーンの良い例
- 「小さな岩を難なく跨ぎ、道の先へ進んだ。」（1文完結、乾いた）
- 「壊れた岩は脆い。手で押すと砂のように崩れた。」（2文、短い）
- 「優しい竜が道を砕いている。古い石畳を壊し、歩きやすい道を作り直してくれているのだ。」（修飾語と述語の化学反応）
- 「壊れた子猫が道を砕いている。ぬいぐるみが道を砕けるわけがない。しかし道はなぜかぼろぼろだ。」（不条理なユーモア）

## 5. 存在シート準拠

- 各outcomeは `existence-sheets.md` の **interpretation** を第一候補として使う
- **alt** はノードの文脈に合う場合のみ使用（括弧内の使い分けヒントを参照）
- 存在シートに書かれていない独自解釈は**原則禁止**
- 同じ存在が複数ノードに出る場合、解釈の**一貫性**を保つ

### 既存ノード(A-E)で確立された解釈（必ず踏襲）
- 壊れた子猫 = ぬいぐるみ、綿が出ている
- 壊れた竜 = 翼は折れ、鱗は剥がれ、火も吐けない
- 壊れた蛇 = 脱皮の途中で力尽きた、古い皮が半分残っている
- 壊れた蝶 = 片羽がちぎれている
- 壊れた騎士 = 鎧はへこみ、剣は折れている。それでも立っている
- 壊れた風 = 方向が定まらない乱気流
- 壊れた炎 = 色がおかしい（緑や紫）。温かくない
- 壊れた影 = 立体的にめくれ上がっている。平面に戻れない
- 壊れた王 = 王冠が歪み、目は虚ろ
- 透き通った子猫 = 硝子の子猫。心臓が透けて見える
- 透き通った蛇 = 飲み込んだものが体内に透けて見える。骨格が美しい
- 透き通った竜 = 水晶でできた竜。体を通して景色が歪んで見える
- 透き通った風 = 澄み渡る風。視界が果てまで開ける
- 透き通った騎士 = 影の薄い騎士。いるのに気づかれない
- 透き通った魔女 = 老いて存在が希薄。向こうが透けて見える
- 呪われた子猫 = 拾った者が捨てられなくなる猫
- 呪われた蛇 = 脱皮するたびに大きくなる
- 呪われた竜 = 守護竜だったが暴走。目に理性がない
- 呪われた影 = 触れたものが影になる
- 呪われた歌 = 一度聴くと頭から離れない
- 堅い子猫 = 陶器の招き猫。目だけが動く
- 燃え盛る子猫 = 炎をまとっているが本猫は気にしていない
- 燃え盛る竜 = 全身が業火。近づくだけで髪が焦げる
- 燃え盛る蝶 = 羽が炎でできている。飛んだ跡に火の粉
- 小さな竜 = 手のひらサイズ。翼を広げても雀ほど
- 小さな蛇 = 指に巻きつくほど細い
- 小さな騎士 = 子供の騎士。大きすぎる兜で前が見えていない
- 小さな王 = 子供が王冠をかぶっている
- 眠い竜 = 宝の山の上で居眠り。いびきが地鳴り

## 6. damage / quill 基準

### バランス環境（2026-04-06改訂）
- HP 15。damage 5 = HPの1/3。2回で瀕死
- 全体AP 12。全ノードで最適解を選べない。ノーチェンジ通過が戦略上必要
- **初期値のoutcomeは damage 2-3 が理想**（ノーチェンジ通過は「そこそこ痛いが致命的ではない」）

### damage
| 状況 | damage |
|------|--------|
| 完全に無害・有益 | 0 または -1（回復） |
| 軽い被害 | 1-2 |
| 初期値のまま（改善なし） | 2-3 |
| 有害だが通過可能 | 3-4 |
| 致命的な脅威 | 5 |

### quill
| 状況 | quill |
|------|-------|
| 改善なし・変化少 | 1-2 |
| まあまあの書き換え | 3-4 |
| うまい書き換え | 5-6 |
| 見事な書き換え | 7-8 |
| 傑作（稀） | 9-10 |

### 判断ガイド
- **脅威低い存在**（子猫, 蝶, 小さな, 優しい等）→ damage低, quill高
- **脅威高い存在**（竜, 炎, 燃え盛る, 鋭い等）→ damage高, quill低
- **修飾語が主語を強める**（燃え盛る炎）→ 効果が極端に
- **修飾語が主語を裏切る**（壊れた竜, 優しい蛇）→ 意外性でquill高め
- **ノードの文脈との相性**も考慮する

## 7. rewardItems

- **全outcomeの15-20%** にrewardItemsを付与
- quill 5以上のoutcomeに付ける傾向
- アイテム名と説明は存在の解釈に紐づける
- sellPrice: 5-20の範囲
- rewardCardsは非常に稀（2-3%以下）

---

## 8. セルフレビューチェックリスト

生成後に必ず確認:
1. ☐ 同じ文型パターンが3回以上連続していないか
2. ☐ 存在シートの解釈から逸脱していないか（特に§5の確立済み解釈）
3. ☐ 格の整合性
4. ☐ damage/quillが基準範囲内か
5. ☐ 修飾語の効果が反映されているか
6. ☐ 同じ主語の異なる修飾語outcomeで変化があるか
7. ☐ テキストに面白さがあるか（テンプレ臭がしないか）
8. ☐ rewardItemsの割合が15-20%か
9. ☐ 既存ノード(A-E)の存在解釈と一貫しているか
10. ☐ 「気がした」「感じた」等の内面描写を使っていないか
11. ☐ 4文以上のテキストがないか
12. ☐ 過去形「〜した」が多用されていないか（行動描写での使用は許容）

## 9. defaultOutcome（歪み）

述語可変ノードでのみ使用。格歪みが発生する組み合わせ全体に1つ:

```
- **defaultOutcome**:
  - text: 文が歪んだ。言葉が世界に馴染まず、宙に散る。体に鈍い痛みが走る。
  - damage: 4
  - quill: 0
```
